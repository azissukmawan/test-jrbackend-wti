Bubble sort memiliki kompleksitas O(n2) karena pada intinya algoritma ini menggunakan,
nested loops untuk mengurutkan data.
kompleksitas runtime algoritma O(n2) berjalan dalam waktu kuadrat.
jadi semakin besar data semakin tidak efisien algoritma nya.
kumpulan data kecil = lumayan oke.
kumpulan data besar = BURUK (jangan dipakai).
